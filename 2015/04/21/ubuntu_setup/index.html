设置oh-my-zsh

  1. 具体的可以查看池建强的博客
  2. sudo apt-get install zsh
  3. 改为默认sh, chsh -s /bin/zsh, 如果不启作用, 我是重启了电脑可以
  4. 安装oh my zsh wget https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh
  5. 自定义配置及使用

安装GIT

  1. GIT现在使用的人非常多, 而且方便过SVN, 并且太多公司和个人都将软件放在github上, 我们还有什么理由不用它呢
  2. 安装非常简单, sudo apt-get install git

  3. 配置你的git用户名和密码, 以及ssh. git config --global color.ui ture git config --global user.name "your name" git config --global user.email "your@email.com ssh-keygen -t rsa -C "your@email.com

  4. 增加到你的SSH里面, git_ssh, cat ~/.ssh/id_rsa.pub ,复制控制台的输出到上面的SSH

  5. 测试是否成功, ssh -T git@github.com, 成功的话会返回XXX! you've successfully

安装Nodejs && npm

使用下面的命令依次安装
sudo apt-get update
sudo apt-get install nodejs
sudo apt-get install npm

安装mysql

sudo apt-get install mysql-client-5.5. mysql-server-5.5, 这里需要注意的是要指定mysql 的版本号, 否则可能出错 , 中间会提示输入root的密码

安装PostgreSQL

sudo apt-get install postgresql-common
sudo apt-get install postgresql-9.3 libpg-dev
